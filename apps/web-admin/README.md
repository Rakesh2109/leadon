# LeadOn Web Admin

Vite, React, Tailwind CSS, and lucide-react power the first admin dashboard MVP.

## Screens

- Login card
- Leader dashboard
- User management
- Team management
- Content management
- Reports
- Settings role switcher

## Run

```bash
cd apps/web-admin
npm install
npm run dev
```

The app uses `VITE_API_BASE_URL`, defaulting to `http://localhost:4000/api`.
